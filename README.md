## EN English

**DasPasswords** is a simple password generator and manager in one application.  
It helps you quickly create strong passwords and store account data locally ‚Äî no cloud services involved.

### Features
- Custom password length
- Character selection: letters, numbers, special symbols
- Custom allowed characters (you can add or remove any symbol, even emojis)
- Built-in password manager
- One-click copy for login and password
- **Mark** field to identify the related website or service
